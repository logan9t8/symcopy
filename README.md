# What is it?
A CLI tool to quickly copy Unicode symbols to clipboard
```
symcopy <keyword>
```

# Installation
1. Install fzf with `winget install junegunn.fzf`
1. Place `symcopy.exe` and `unicode_ds.json` in a (same) directory included in the PATH
